Opis szczegółowy

Reprezentuje statystyki użytkownika w grze "Kółko i Krzyżyk".

Klasa przechowuje szczegółowe informacje o wynikach użytkownika, w tym liczbę rozegranych gier, wygranych gier, przegranych gier, gier przeciwko komputerowi oraz gier lokalnych.
