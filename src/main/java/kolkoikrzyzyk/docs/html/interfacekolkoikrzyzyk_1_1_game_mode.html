Opis szczegółowy

Interfejs definiujący tryb gry w "Kółko i Krzyżyk".

Interfejs ten określa podstawowe metody, które muszą być zaimplementowane przez każdą klasę obsługującą różne tryby gry, takie jak gra lokalna, gra przeciwko botowi czy gra online.

Dokumentacja funkcji składowych

◆ makeMove()

void kolkoikrzyzyk.GameMode.makeMove ( int row,
int col )

Wykonuje ruch gracza w określonej pozycji na planszy.

Parametry
rowWiersz planszy (0-2).
colKolumna planszy (0-2).

Metoda realizuje logikę wykonania ruchu w wybranym polu planszy. W zależności od implementacji może uwzględniać lokalną grę, boty lub tryb online.

Implementowany w kolkoikrzyzyk.ClientGame, kolkoikrzyzyk.LocalGame, kolkoikrzyzyk.ServerGame i kolkoikrzyzyk.VsComputerGame.

◆ resetGame()

void kolkoikrzyzyk.GameMode.resetGame ( )

Resetuje stan gry do początkowego.

Przywraca planszę do stanu początkowego, usuwając wszystkie symbole oraz resetując wewnętrzny stan gry. Może być używane po zakończeniu gry, aby rozpocząć nową rozgrywkę.

Implementowany w kolkoikrzyzyk.GameBase.
