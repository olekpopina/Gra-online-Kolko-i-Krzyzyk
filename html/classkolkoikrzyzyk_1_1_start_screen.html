Opis szczegółowy

Ekran startowy gry "Kółko i Krzyżyk".

Klasa odpowiedzialna za wyświetlanie ekranu startowego, na którym użytkownik może zobaczyć swoje dane (jeśli jest zalogowany), najlepszych graczy oraz wybrać opcje gry.

Nota
Ekran wyświetla informacje o zalogowanym użytkowniku oraz najlepszych graczach w grze.

Konstruktor ekranu startowego.

Inicjalizuje ekran startowy z odpowiednim tytułem, który zawiera imię zalogowanego użytkownika. Jeśli użytkownik nie jest zalogowany, wyświetla się "Gość".

◆ showChangePasswordDialog()

void kolkoikrzyzyk.StartScreen.showChangePasswordDialog ( )
private

Wyświetla okno dialogowe zmiany hasła.

Okno dialogowe umożliwia wprowadzenie starego hasła, nowego hasła oraz potwierdzenia nowego hasła. Jeśli nowe hasła nie są zgodne, użytkownik otrzymuje komunikat o błędzie. W przypadku poprawnego wprowadzenia, hasło jest zmieniane, a użytkownik otrzymuje stosowny komunikat.

◆ showRegisterDialog()

void kolkoikrzyzyk.StartScreen.showRegisterDialog ( )
private

Wyświetla okno dialogowe rejestracji nowego użytkownika.

Okno dialogowe umożliwia wprowadzenie nazwy użytkownika oraz hasła. Po zatwierdzeniu, jeśli użytkownik nie istnieje w bazie danych, konto jest tworzone, a użytkownik otrzymuje stosowny komunikat. W przeciwnym razie pojawia się komunikat o błędzie, informujący, że użytkownik już istnieje.

◆ showUserStats()

void kolkoikrzyzyk.StartScreen.showUserStats ( )
private

Wyświetla statystyki użytkownika.

Jeśli użytkownik jest zalogowany, wyświetlane są szczegółowe statystyki jego gier, takie jak liczba rozegranych gier online, wygranych gier, przegranych gier oraz stosunek wygranych do rozegranych gier. Pokazywane są również statystyki gier przeciwko komputerowi oraz gier lokalnych.

Jeśli użytkownik nie jest zalogowany lub nie udało się pobrać statystyk z bazy danych, wyświetlany jest odpowiedni komunikat o błędzie.
