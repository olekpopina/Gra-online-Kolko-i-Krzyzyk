◆ resetGame()

void kolkoikrzyzyk.GameBase.resetGame ( )

Resetuje grę, czyszcząc planszę i stan gry.

Jeśli użytkownik jest zalogowany, aktualizuje statystyki w bazie danych w oparciu o wynik ostatniej gry.

Nota
Metoda wykonuje reset gry, przywracając wszystkie pola do stanu początkowego.

◆ returnToMainMenu()

void kolkoikrzyzyk.GameBase.returnToMainMenu ( )
protected

Powraca do głównego menu gry po potwierdzeniu przez użytkownika.

Wyświetla okno dialogowe z pytaniem o potwierdzenie. Jeśli użytkownik zaakceptuje, zamyka aktualne okno gry i otwiera ekran startowy. Jeśli użytkownik odmówi, pozostaje w grze.

Reimplementowana w kolkoikrzyzyk.ClientGame i kolkoikrzyzyk.ServerGame.
